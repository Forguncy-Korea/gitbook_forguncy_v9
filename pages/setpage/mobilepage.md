# 모바일 페이지 만들기

모바일 페이지는 모바일 브라우저에서 액세스하는 데 사용되는 페이지입니다. 모바일 애플리케이션에서 사용하기 위해서는 포건시에서 제공하는 앱을 사용하여야 합니다.

## 모바일 페이지 만들기

다음과 같이 모바일 브라우저 액세스에 사용할 페이지를 만듭니다.

* 방법 1. 리본 메뉴에서 \[만들기]>\[모바일 페이지]를 선택합니다.

![모바일 페이지 ](<../../.gitbook/assets/333 (1).png>)

* 방법 2. 개체 관리자의 페이지 탭에서 마우스 오른쪽 버튼을 클릭하여 새 모바일  페이지 만들기를 클릭합니다.

생성이 완료되면 \[홈]>\[실행]버튼을 릭하거나 빠른 액세스 막대를 클릭하면 컴퓨터 브라우저의 모바일 시뮬레이터에 만든 모바일 페이지가 표시됩니다.

{% hint style="info" %}
* 모바일  시뮬레이터에 모바일  페이지를 표시하지 않으려면 드롭다운 메뉴에서 브라우저 실행을 선택할 수 있습니다. 모바일 시뮬레이터를 사용하여 일반 웹 페이지 페이지를 실행하도록 선택하면 설정된 휴대폰 페이지가 표시됩니다.
* 일반 페이지와 모바일 페이지로 나누어져 있지만 컴퓨터 브라우저 또는 모바일 브라우저에서 일반 페이지에 액세스할 수 있습니다(휴대폰 페이지가 설정되지 않은 경우).
* 모바일 브라우저에서 페이지에 액세스할 때 설정된 일부 셀 서식이 작동하지 않을 수 있습니다. 예를 들어 많은 모바일 브라우저는 하나의 글꼴만 지원합니다.
{% endhint %}

모바일 페이지와 일반 페이지의 차이점은 아래 표에 나와 있습니다.

| 아이콘        | 모바일 페이지의 아이콘은 일반 페이지의 아이콘과 다릅니다.![](https://help.grapecity.com.cn/download/thumbnails/72352436/image2017-7-24_14-14-10.png?version=1\&modificationDate=1648092531000\&api=v2)![](https://help.grapecity.com.cn/download/thumbnails/72352436/image2017-7-24_14-14-46.png?version=1\&modificationDate=1648092531000\&api=v2) |
| 셀 타입       | <p>다른 환경에서 쉽게 사용할 수 있도록 동일한 셀 타의 모양과 동작은 모바일 브라우저와 컴퓨터 브라우저 간에 약간 다를 수 있습니다.</p><p>따라서 속성 설정 영역의 셀 설정 영역에서 일부 셀 유형에 대한 설정 옵션은 모바일 페이지와 일반 페이지 간에 약간 다릅니다.</p>                                                                                                                                                              |

| 테이블        | <p>모바일 브라우저에서는 테이블의 데이터를 삭제할 수 있지만 테이블에서 편집하고 추가할 수는 없습니다.</p><p>따라서 모바일 페이지에서는 테이블을 편집 허용, 새 행 추가 허용, 드롭다운 값 허용 및 선택 영역의 위치로 드래그할 수 있도록 설정할 수 없습니다.</p>                                                                                                                                                                   |
