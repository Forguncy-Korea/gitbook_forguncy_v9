# 데이터로드 설정



포건시에서 데이터 로드에 대한 몇 가지 설정이 있습니다.

* 페이지를 표시할 때 데이터를 불러오지 않습니다&#x20;
* 자동으로 데이터 다시 불러오기&#x20;
* 오토스크롤

![](<../../.gitbook/assets/image (1038).png>)

## 페이지를 표시할 때 데이터를 불러오지 않습니다.&#x20;

기본적으로 선택되어 있지 않은 상태입니다.&#x20;

리스트뷰를 선택하고 마우스 오른쪽 버튼 클릭, 오른쪽 클릭 메뉴에서 "리스트뷰 옵션 설정"을 선택, 리스트뷰 설정 대화 상자를 팝업창에서 데이터"탭을 선택하고, "페이지를 표시할 때 데이터를 불러오지 않습니다"를 선택하 페이지를 열 때 테이블 데이터를 로드하지 않습니다, 데이터가 새로 고쳐질 때만 데이터가 로드됩니다.

![](<../../.gitbook/assets/image (1271).png>)

## 자동으로 데이터 다시 불러오기&#x20;

기본적으로 "자동으로 데이터 다시 불러오기 "를 선택되어 있지 않습니다.&#x20;

리스트뷰를 선택하고 마우스 오른쪽 버튼 클릭, 오른쪽 클릭 메뉴에서 "리스트뷰 옵션 설정"을 선택, 리스트뷰 설정 대화 상자를 팝업창에서 데이터"탭을 선택하고, \[자동으로 데이터 다시 불러오기] 선택한 후 데이터를 새로 고치는 간격을 설정할 수 있습니다.

예를 들어 간격을 1초로 설정하면 데이터가 1초 후에 새로 고쳐집니다.

![](<../../.gitbook/assets/image (1172).png>)

## 오토스크롤&#x20;

오토 스크롤을 사용하여 리스트뷰에서 데이터를 자동으로 스크롤합니다.

![](<../../.gitbook/assets/image (758).png>)

리스트뷰를 선택하고 마우스 오른쪽 버튼 클릭, 오른쪽 클릭 메뉴에서 "리스트뷰 옵션 설정"을 선택, 리스트뷰 설정 대화 상자를 팝업창에서 데이터"탭을 선택하고,  \[오토스크롤]을 선택하면 자동 스크롤 속도(느린, 중간 및 빠른)를 설정할 수 있습니다.

리스트뷰의 자동 스크롤은 픽셀 단위로 스크롤됩니다.
