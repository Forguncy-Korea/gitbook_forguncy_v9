# 날짜

셀을 날짜 유형으로 설정하고 실행 후 날짜를 직접 입력하거나 달력에서 날짜를 선택할 수 있습니다. 날짜 입력 상자는 다음 그림에 나와 있습니다.

<figure><img src="../../.gitbook/assets/image (1898).png" alt=""><figcaption></figcaption></figure>

## 날짜 셀 설정&#x20;

날짜 입력 상자를 선택하고 속성 설정 영역에서 셀 설정 탭을 선택합니다.



| 설정                   | 설명                                                                                                                                                                                                                                                                    |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 명령 편집                | 값이 변경될 때 실행되는 명령을 설정합니다. [명령](../../command/command/) 참조하십시오.                                                                                                                                                                                                         |
| 데이터 검증               | 페이지 측 유효성 검사에서 확인할 수 있는 날짜 입력 상자의 데이터 유효성 검사를 설정합니다.                                                                                                                                                                                                                  |
| 기본값                  | <p>날짜 입력 상자의 기본값을 지정하고 수식을 입력하면 수식의 결과가 기본값으로 사용됩니다. 예를 들면 다음과 같습니다.</p><ul><li>기본값을 오늘로 설정하고 기본값을 공식 =TODAY()로 설정합니다.</li><li>기본값을 내일로 설정하고 기본값을 수식 =TODAY()+1로 설정합니다.</li><li>기본값을 어제로 설정하고 기본값을 공식 =TODAY()-1로 설정합니다.</li></ul>                                    |
| 날짜 형식                | <p>날짜의 편집 형식을 지정합니다.</p><p><img src="../../.gitbook/assets/image (98).png" alt=""></p><p><strong>설명</strong></p><ul><li><strong>날짜의 표시 서식을 지정하려면 셀을 선택하여 셀 서식을 지정하여 날짜의 표시 형식을 지정할 수 있습니다.</strong><br></li><li><strong>이 설정은 모바일 페이지에서 지원되지 않습니다.</strong></li></ul> |
| 드롭다운 버튼 표시 여부        | <p>날짜를 표시할 수 있는 버튼을 표시합니다.</p><p><strong>설명</strong></p><p><strong>이 설정은 모바일 페이지에서 지원되지 않습니다.</strong></p>                                                                                                                                                            |
| 달력  스크롤 막대 표시        | <p>드롭다운 상자에 스크롤 막대를 표시하고 스크롤 막대를 드래그하여 날짜를 빠르게 찾습니다.</p><p></p>                                                                                                                                                                                                       |
| 읽기 전용                | 이 옵션을 선택하면 날짜를 입력하거나 선택할 수 없습니다.                                                                                                                                                                                                                                      |
| 셀 선택 시 전체 선택 상태로 표시  | 이 옵션을 선택하면 날짜 입력 상자에 포커스가 있으면 텍스트 내용이 모두 선택되고 텍스트 내용을 직접 편집할 수 있습니                                                                                                                                                                                                    |



**마우스 휠 및 키보드 위쪽 및 아래쪽 키 동작 설정입니다**

마우스 휠과 키보드 위/아래 키를 사용하여 날짜 셀의 값을 수정할 수 있는지 여부를 설정할 수 있습니다.

리본 메뉴에서 파일을 선택한 다음 옵-> 응용 프로그램 설정을 선택하고 다른 설정 영역에서 마우스 휠 및 키보드 위쪽 및 아래쪽 키 동작 설정 하이퍼링크를 클릭하여 대화 상자를 표시합니다.

대화 상자에서 날짜 이후의 마우스 휠 및 키보드의 위쪽 및 아래쪽 키에 대한 확인란을 선택하거나 선택 취소할 수 있습니다.

* 체크박스를 선택하면 실행 후 마우스 휠과 키보드 위 또는 아래로 키를 사용하여 실행 후 날짜 셀의 값을 수정할 수 있습니다.
* 체크박스를 선택하지 않으면 실행 후 마우스 휠과 키보드 위/아래 키를 사용하여 날짜 셀의 값을 수정할 수 없습니다.
