# 체크박스

셀을 체크박스으로 설정하고 실행 후 체크박스를 선택하거나 선택 취소할 수 있습니다.&#x20;

체크박스를 선택하면 값이 "1"이고 데이터베이스에 "1"이 저장됩니다. 체크박스의 선택을 취소하면 값이 "0"이고 데이터베이스에 "0"으로 저장됩니다.

<figure><img src="../../.gitbook/assets/image (1004).png" alt=""><figcaption></figcaption></figure>

| 설정          | 설명                                                                                                                               |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------- |
| 편집 명령       | 확인란을 선택하거나 선택하지 않은 경우 실행할 명령을 설정합니다. 명령은 [명령 ](../../command/command/)을 참조하십시오 .                                                 |
| 데이터 검증      | 확인란의 데이터 유효성 검사를 설정합니다. 자세한 내용은 [페이지 측 유효성 검사](../../pages/verification_data/page/trigger.md)를 참조하십시오 .                          |
| 선택된 상태로 표시  | 확인란의 시작 상태를 설정합니다.                                                                                                               |
| 표시 텍스트      | <p>확인란의 오른쪽에 표시 텍스트를 설정합니다. 예를 들어, 표시된 텍스트는 다음 그림과 같이 "완료여부"입니다.</p><p><img src="/broken/files/raYXCeBt5XFY0OUdEFBQ" alt=""></p> |
| 스타일         | 시스템, 모던, 스위치, 버튼의 네 가지 스타일을 사용할 수 있습니다. 현대적인 스타일을 기본으로 합니다.                                                                      |
| 주요 표시 색상    | 스타일이  체크 박스의 기본 색상을 설정할 수 있습니다.                                                                                                  |
| 선택 표시 색상    | 스타일이  체크 박스의 보조 색상을 설정할 수 있습니다.                                                                                                  |
| 읽기 전용       | 이 항목을 체크한 후에는 체크박스 상태를 수정할 수 없습니다.                                                                                               |
| 편집시에만 표시    | 리스트뷰 행의 셀을 체크박스로 설정할 때 설정할 수 있습니다. 체크 후 셀이 편집 상태일 때만 셀의 종류가 표시됩니다.                                                               |
