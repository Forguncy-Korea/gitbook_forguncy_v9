# 셀 유형



셀 유형은 셀에 렌더링되는 정보의 유형과 해당 정보가 표시되는 방식 및 사용자가 셀과 상호 작용하는 방식을 정의합니다.

현재 포건시에서 지원하는 셀 유형에는 버튼, 텍스트 상자, 여러 줄 텍스트 상자, 체크박, 드롭다운, 라디오 버튼, 하이퍼링크, 콤보 박스, 숫자, 날짜, 시간, 그림, 이미지 업로드, 첨부 파일, 사용자 목록 선택기, 데이터 탐색 버튼, 페이징 탐색 버, 프로세스 바, 로그인 사용자, 바코드 등이 포함됩니다.

## 셀 유형 설정 방법&#x20;

셀 범위를 선택하거나 셀 유형에서 선택하거나 드롭다운하여 원하는 셀 유형을 선택합니다.
