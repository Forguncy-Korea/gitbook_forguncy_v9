# 대기명령 (SleepCommand)

타사 인터페이스를 호출할 때 연결 시간 초과 또는 오류를 반환하는 경우 프로그램을 잠시 기다렸다가 다시 요청해야 할 수 있습니다.&#x20;

### 플러그인 다운로드&#x20;

버전에 맞는 플러그인을 다운로드 합니다.

### 사용방법&#x20;

1. 플러그인을 다운로드합니다.
2. Forguncy Builder에서 설치하고 Forguncy Builder를 다시 실행합니다.
3. 서버 명령에서 "대기 명령"을 선택하고 대기 시간만 설정합니다.&#x20;
