# 추적 프로세스 상태

추적 프로세스 상태 명령을 사용하면 성능 문제를 조사할 수 있습니다.&#x20;

많은 고객이 성능 문제에 직면해있습니다. 이 명령을 사용하여 현재 머신의 실행 상태를 출력할 수 있습니다.

### 플러그인 다운로드&#x20;

버전에 맞는 플러그인을 다운로드 합니다.

### 사용방법&#x20;

시스템 성능에 영향을 미칠 수 있는 명령 사이에 추적 프로세스 상태 명령을 추가합니다.  첫번째로그는   "페이지 PDF로 내보내기"  명령을 실행 하기 전의 로그이고, 마지막 로그는 "페이지 PDF로 내보내기"  명령을 실행 하기 후의 로그입니다.
