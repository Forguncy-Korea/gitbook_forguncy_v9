# 오프라인 배포

포건시 빌더는 웹 사이트를 포건시 서버에 직접 게시할 수 있습니다. 그러나 때로는 포건시빌더가 네트워크를 통해 포건시 서버에 직접 연결할 수 없으며 웹 사이트를 오프라인으로 게시 할 수 있습니다.

## 배포 가능한 파일을 저장&#x20;

만든 응용 프로그램을 포건시에서 직접 게시할 수 있는 파일 형식(.fgko)로 저장합니다.

빌더의 리본 메뉴 모음에서 배포-> 배포파일생성을 선택하여 파일을 로컬로 저장합니다.

![](<../../.gitbook/assets/image (1170).png>)

## 오프라인으로 웹사이트를 배포&#x20;

저장된 배포 가능한 파일(.fgko 파일)을 포건시 서버가 있는 컴퓨터에 복사하여 웹 사이트를 오프라인으로 게시할 수 있습니다.

포건시 서버의 설치 경로에서 "OfflinePublishTool .exe"을 찾아 두 번 클릭하여 설치 없이 게시 설정 창을 직접 엽니다.

![](<../../.gitbook/assets/image (1744).png>)

다음 설정을 설정한 후 확인을 클릭하여 사이트를 배포합니다.



| 설정                                   | 설명                                                                                                                     |
| ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------- |
| 파일 경로                                | 오프라인 게시 파일(.fgko 파일)을 선택합니다.                                                                                           |
| 응용 프로그램 이름                           | 배포된 사이트 이름이며, 서비스에 이름이 같은 앱이 이미 있는 경우 새로 게시된 앱이 이전 사이트를 덮어씁니다.                                                         |
| 포트                                   | 기본값은 80입니다. 80 포트를 다른 응용 프로그램에서 사용할 수 없는 경우 1025에서 65534 사이의 값 범위로 다른 포트 번호를 선택할 수 있습니다.                               |
| HTTPS 웹 사이트                          | HTTPS 웹 사이트로 게시해야 하는 경우 Https 게시의 "HTTPS 설정"을 참조하여 포건시 서버에 대한 HTTPS 설정을 미리 구성해야 합니다.                                   |
| 데이터베이스 덮어쓰기                          | 게시된 사이트가 포건시 관리 콘솔에 이미 있는 경우 이 옵션은 원래 사이트의 데이터베이스를 동시에 덮어쓸지 여부를 결정합니다. 사이트를 처음 게시하는 경우 이 옵션을 선택했는지 여부에 관계없이 결과가 일관됩니다. |
| 로컬의 페이지 접근 권한과 UI 사용 권한 설정을 서버에 덮어쓰기 | 이 옵션을 선택하면 빌더에서 설정한 페이지 및 셀 권한이 서버에 게시되고 서버 쪽 페이지 및 셀 사용 권한이 재정의됩니다. 자세한 내용은 페이지 및 셀 권한을 참조하십시오.                       |

## 오프라인으로 패키지를 배포하는 암호&#x20;

웹 사이트를 오프라인으로 배포할 때 서버가 게시자에게 암호를 제공하도록 오프라인 게시 파일의 암호를 설정합니다.

{% hint style="info" %}
&#x20;오프라인 게시 파일은 포건시빌더를 사용하지 않고 포건시 서비스 관리자에서 직접 게시할 수 있습니다. 따라서 암호를 설정한 응용 프로그램은 게시 후 3일 동안만 사용할 수 있으므로(게시자는 암호가 없음) 이러한 방식으로 평가판 웹 사이트를 게시할 수 있습니다.
{% endhint %}

파일->옵션-> 응용프로그램 설정을 선택하여 기타 영역에서 오프라인 배포 암호를 설정합니다.
