# 역할 만들기

사용자 역할은 관리자 역할, 임시 작업자 역할, 감사 역할 등과 같은 구성 가능한 사용자 클래스를 나타내는 병렬 구조(상위 및 하위 관계 없음)입니다.

관리 콘솔의 사용자 관리 영역에서 역할을 클릭하여 모든 역할을 표시합니다. 검색 상자에 키워드를 입력하여 역할을 검색합니다.

![](<../../.gitbook/assets/image (1047).png>)



## 역할 추가하기&#x20;

역할 목록 위에서 새 역할 추가를 클릭하고 팝업 대화 상자에 역할 이름을 입력합니다. 예를 들어 역할 "사업부 관리자"를 추가하면 역할 목록에 표시됩니다.

![](<../../.gitbook/assets/image (1153).png>)

## 역할 관련 작업&#x20;

### 역할에 멤버 추가하기&#x20;

역할을 추가한 후에는 역할에 구성원을 추가해야 합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72356663/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1648092595000\&api=v2) 역할을 선택하고 오른쪽에서 멤버 추가를 클릭합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72356663/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1648092595000\&api=v2) 팝업 멤버 편집 대화 상자에서 사용자 인증 모드를 선택하고 검색 상자에 검색 키워드를 입력하고 Enter 키를 클릭하거나 눌러 검색하거나 사용자를 직접 선택할 수 있습니다.![](https://help.grapecity.com.cn/download/thumbnails/72356663/image2018-2-7_11-52-55.png?version=1\&modificationDate=1648092595000\&api=v2)

![](<../../.gitbook/assets/image (1649).png>)

{% hint style="info" %}
여기서 선택한 사용자 인증 모드는 사용자를 만들 때와 일치해야 합니다.

* 사용자를 만들 때  Foguncy 인증 사용자를 선택한 경우 여기에서 일반 계정을 선택합니다.
* 사용자를 만들 때 Windows 인증 사용자를 선택한 경우 여기에서 Windows 도메인 사용자를 선택합니다.
{% endhint %}

![](https://help.grapecity.com.cn/download/thumbnails/72356663/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1648092595000\&api=v2) 확인을 클릭하면 사용자를 성공적으로 추가할 수 있습니다. 사용자가 서명한 체크박스를 여러 번 검색하고 선택한 모든 사용자를 한 번에 추가하여 사용자를 추가할 수 있습니다.

### 역할 및 권한

&#x20;역할의 권한 관계에서 현재 역할의 역할 권한을 조회/설정할 수 있으며 현재 역할 권한을 가진 역할이 나열됩니다.&#x20;

#### 다음 역할 권한을 가집니다.&#x20;

&#x20; ![](<../../.gitbook/assets/image (1699).png>)버튼을 클릭 한 후 , 팝업 역할 설정 창에서 역할을 선택합니다. 이는 현재 역할이 선택한 역할의 권한을 가짐을 의미합니다.&#x20;

<figure><img src="../../.gitbook/assets/image (1145).png" alt=""><figcaption></figcaption></figure>

**역할 권한 관계 다이어 그램**&#x20;

역할을 선택한 후 역할의 권한 관계에서 역할의 권한 관계 다이어그램을 볼 수 있으며 어떤 역할이 현재 역할의 권한을 가지고 있는지, 현재 역할이 어떤 권한을 가지고 있는지 확인할 수 있습니다.

<figure><img src="../../.gitbook/assets/image (1936).png" alt=""><figcaption></figcaption></figure>

### 조직 노드&#x20;

역할은 조직 노드 추가를 지원하여 조직 노드의 구성원이 이 역할에 속함을 나타냅니다. 조직 노드를 선택할 때 이 노드의 구성원 또는 이 노드 및 해당 하위 노드의 모든 구성원을 선택할 수 있습니다.

역할 선택 후 오른쪽 기본 정보의 조직 노드 영역에서 조직 노드 수정을 클릭합니다.
