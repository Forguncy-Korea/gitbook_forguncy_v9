# 사용자 정보 보기

사용자 정보 보기를 사용하면 사용자 계정 관리 플랫폼의 모든 사용자와 정보를 볼 수 있습니다.

## 사용자 정보 보기 만들기&#x20;

리본 메뉴 모음에서 \[만들기>뷰>사용자 정보 뷰 만들기]를 선택합니다.

![](<../../.gitbook/assets/image (1574).png>)

작성이 완료되면 테이블 탭 막대 아래에 사용자 정보 뷰가 표시됩니다. 여기에는 사용자 계정 관리 플랫폼의 모든 사용자 및 정보가 포함됩니다.

![](<../../.gitbook/assets/image (1143).png>)



여기서:

* 사용자 이름, 전체 이름, 메일은 사용자의 기본 속성이며 전화는 사용자 지정 속성입니다.
* 사용자에게 둘 이상의 역할이 있는 경우 여러 역할이 쉼표로 구분됩니다.
* 조직 상위 항목 사용 ‖ 사용자가 여러 조직 상위를 가지고 있는 경우 여러 조직 상위 간에 | 사용합니다. 분리;
* 회사 및 부서는 조직 수준입니다. 조직 구조의 설정에 따라 사용자가 있는 조직 구조를 표시합니다.
