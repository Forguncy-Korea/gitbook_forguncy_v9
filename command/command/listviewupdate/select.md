# 선택&#x20;

리스트뷰 업데이트명령에서 작업 유형이 "선택"이면 선택 범주 아래에 8개의 동작 유형이 있습니다.

<figure><img src="../../../.gitbook/assets/image (1009).png" alt=""><figcaption></figcaption></figure>

## 현재 행 번호 가져오기&#x20;

현재 행의 번호를 가져옵니다. 현재 행 번호를 가져온 후 반환 값을 후속 명령에서 사용할 수 있는 변수에 저장할 수 있습니다.

<figure><img src="../../../.gitbook/assets/image (156).png" alt=""><figcaption></figcaption></figure>

## 현재 열 번호 가져오기&#x20;

현재 열의 번호를 가져옵니다. 현재 열 번호를 가져온 후 반환 값을 후속 명령에서 사용할 수 있는 변수에 저장할 수 있습니다.

<figure><img src="../../../.gitbook/assets/image (1310).png" alt=""><figcaption></figcaption></figure>

## 선택한 행 번호 목록 가져오기&#x20;

선택한 행의 번호목록을 가져옵니다. 선택한 행 인덱스 목록을 가져온 후 반환 값을 후속 명령에서 사용할 수 있는 변수에 저장할 수 있습니다.

<figure><img src="../../../.gitbook/assets/image (600).png" alt=""><figcaption></figcaption></figure>

## 셀 선택&#x20;

특정 셀을 선택하려면 선택하려는 셀의 행 번호와 열 번호/열 이름을 설정해야 합니다.

<figure><img src="../../../.gitbook/assets/image (1969).png" alt=""><figcaption></figcaption></figure>

## 선택한 모든 행 지우기&#x20;

리스트뷰에서 선택한 모든 행을 지웁니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (1062).png" alt=""><figcaption></figcaption></figure>

## 모든 행 선택

리스트뷰의 모든 행을 선택합니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (1514).png" alt=""><figcaption></figcaption></figure>

## 선택 행 추가&#x20;

선택한 행을 추가하려면 추가할 선택된 행의 행 번호를 설정해야 합니다.

<figure><img src="../../../.gitbook/assets/image (593).png" alt=""><figcaption></figcaption></figure>

## 행 선택 제거&#x20;

행 선택을 취소하려면 선택 해제할 행의 행 번호를 설정해야 합니다.

<figure><img src="../../../.gitbook/assets/image (885).png" alt=""><figcaption></figcaption></figure>
