# 리스트뷰의 열 속성 변경하기

리스트뷰 열이 많은 경우 페이지에 열을 완전히 표시하기가 어렵습니다. 열 옵션 명령을 사용하여 리스트뷰에 없는 열을 추가하거나 리스트뷰에  이미 있는 열을 삭제할 수 있으며 언제든지 표시할 열을 사용자 지정하여 데이터를 쉽게 볼 수 있습니다.

![](<../../.gitbook/assets/image (1256).png>)

예를 들어 페이지의 테이블에는 많은 열이 있고, 페이지에 테이블을 완전히 표시할 수 없으며, 페이지 아래에 스크롤 막대가 나타나고, 모든 데이터를 보려면 스크롤 막대를 이동해야 하며, 이러한 작업은 사용자 환경에 영향을 줄 수 있습니다.

이제 열 옵션 명령을 사용하여 테이블에 표시되는 열을 사용자 지정하고 보려는 열을 표시할 수 있습니다.

![](https://help.grapecity.com.cn/download/thumbnails/72354479/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1648092564000\&api=v2) 열 옵션 명령을 사용하기 전에 테이블의 열 이름을 설정해야 하며, 새로운 자동 열 이름 지정 기능을 사용하여 테이블의 각 열에 대한 열 이름을 빠르게 설정할 수 있습니다.

테이블을 선택한 후 마우스 오른쪽 버튼 클릭하고 마우스 오른쪽 버튼 클릭 메뉴에서 \[열 이름 자동 설정]을 선택합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72354479/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1648092564000\&api=v2) 테이블의 일부 열을 삭제하여 테이블 영역을 작게 만듭니다.

![](https://help.grapecity.com.cn/download/thumbnails/72354479/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1648092564000\&api=v2) 페이지에 버튼을 추가하고 명령을 열 옵션 명령으로 선택합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72354479/%E6%AD%A5%E9%AA%A44.png?version=1\&modificationDate=1648092564000\&api=v2) 열 옵션 명령을 설정하고 대상 테이블을 선택하면 테이블에 없는 필드가 열 설정에 자동으로 채워집니다.

![](<../../.gitbook/assets/image (1987).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72354479/%E6%AD%A5%E9%AA%A45.png?version=1\&modificationDate=1648092564000\&api=v2) 열 설정에 열을 추가하거나, 열을 삭제하거나, 열을 위로 이동하고, 열을 아래로 이동하거나, 대상 열, 이름, 너비, 열 머리글 스타일 및 셀 스타일을 직접 변경할 수 있습니다.

예를 들어 범주 이름 열의 너비를 150으로 설정하고 열 헤더의 채우기를 노란색으로 설정하고 셀의 채우기를 빨간색으로 설정합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72354479/%E6%AD%A5%E9%AA%A46.png?version=1\&modificationDate=1648092564000\&api=v2) 설정이 완료되면 실행하고 열 옵션 버튼을 클릭하고 열 옵션 설정 대화 상자에서 사용 가능한 열 영역에서 열을 선택하고 클릭하여 표시 열 영역으로 이동합니다.![](https://help.grapecity.com.cn/download/thumbnails/72354479/image2019-3-18_11-2-33.png?version=1\&modificationDate=1648092564000\&api=v2)

예를 들어 범주 이름 열을 선택하여 표시 열로 이동합니다.

![](<../../.gitbook/assets/image (1488).png>)

표시 열에서 열을 선택하고 클릭하여 표시 열의 열을 사용 가능한 열로 다시 이동합니다.![](https://help.grapecity.com.cn/download/thumbnails/72354479/image2019-3-18_11-6-38.png?version=1\&modificationDate=1648092564000\&api=v2)

오른쪽 또는 을 클릭하여 테이블에 열이 표시되는 순서를 조정합니다.![](https://help.grapecity.com.cn/download/thumbnails/72354479/image2019-3-18_11-4-45.png?version=1\&modificationDate=1648092564000\&api=v2)![](https://help.grapecity.com.cn/download/thumbnails/72354479/image2019-3-18_11-4-57.png?version=1\&modificationDate=1648092564000\&api=v2)

예를 들어 다음 그림과 같이 범주 이름 열을 제품 이름 아래로 이동하고 고객 전화 열을 사용 가능한 열로 다시 이동합니다.

설정이 완료되면 확인을 클릭하면 페이지에 다음과 같이 표시됩니다.
