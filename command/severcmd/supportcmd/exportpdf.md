# 페이지 PDF로 내보내기

서버단 명령에서 페이지 PDF로 내보내기 명령을 사용하여 서버 측에서 페이지를 내보낸 다음 인쇄할 수 있습니다.

다음은 서버단 명령에서 PDF 명령으로 페이지를 인쇄/내보내는 방법에 대한 자세한 설명입니다.

**절차를 따르십시오**

1. 개체 관리자의 서버단 명령 탭을 마우스 오른쪽 버튼을 클릭하고 서버단 명령 만들기를 선택하여 서버 단 명령 만들기 대화 상자를 나타납니다.\
   또는 폴더 만들기를 선택하여 폴더에 서버 명령을 만듭니다.

![](<../../../.gitbook/assets/image (1680).png>)

리본 메뉴 모음에서 만들기를 클릭하고 서버 개체 영역에서 서버 명령을 클릭하여 서버단 명령 만들기 대화 상자를 팝업할 수도 있습니다.

2. &#x20;서버 명령의 일반 설정을 편집합니다. 설정 명령의 이름은 \[PDF 내보내기]입니다.
3. &#x20; 서버 명령을 편집하는 명령입니다. \[편집 명령] 하이퍼링크를 클릭하고 서버 명령 편집 대화 상자를 표시하고 페이지 PDF로 내보내기 선택합니다.

* 페이지 내보내기: 앱에서 페이지를 선택하여 PDF로 내보냅니다.
* 내보내기 파일 경로: PDF 파일 저장소를 내보내는 경로를 설정하고 "D:\PDF 내보내기\주문 목록 .pdf"와 같은 파일의 전체 경로를 입력해야 하며 폴더는 기존 폴더여야 하며 접미사 이름 ".pdf"은 생략할 수 있습니다.
* 행 높이 자동 조정 : 텍스트 내용이 너무 많으면 행 높이를 자동으로 조정하여 모든 텍스트가 표시되도록 합니다.
* 고급 설정: 내보낸 페이지에서 내보내기를 위해 페이지에 입력한 값을 서버로 전달해야 하는 경우 고급 설정이 필요합니다. 페이지를 Excel로 내보내기 명령의 고급 설정과 동일한 방식으로 사용됩니다.

![](<../../../.gitbook/assets/image (1111).png>)

4. 설정 완료되면 이 서버단 명령을 호출할 수 있습니다.

예를 들어 페이지에서 셀 범위를 선택하고 버튼으로 설정합니다. 버튼의 명령을 편집하고 "서버단 명령 호출"으로 명령을 선택한 다음 서버단 명령 뒤에 있는 드롭다운을 클릭하고 드롭다운 목록에서 "PDF 내보내기"를 선택합니다.

![](<../../../.gitbook/assets/image (194).png>)

5. 설정이 완료되면 \[확인]을 클릭하여 대화 상자를 닫습니다. 페이지를 실행합니다.페이지에서 \[PDF 내보내기] 버튼을 클릭하면 설정된 지정된 경로 아래에 내보낸 PDF 파일이 표시됩니다.
