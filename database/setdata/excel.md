# Excel 파일로 데이터 테이블 만들기

포건시는 Excel파일의 데이터를 가져와 데이터 테이블을 생성할 수 있도록 지원합니다. 예를 들어 포건시에서 다음 그림과 같이 Excel 파일을 사용하여 데이터 테이블을 만듭니다.

![엑셀 파일 ](<../../.gitbook/assets/Excel file (1).png>)

## Excel 파일로 데이터 테이블 만들기 &#x20;

위의 그림 Excel 파일을 예로 들어 포건시에서 Excel 파일에서 데이터 테이블을 만드는 방법을 보여 드리겠습니다.

아래 절차대로 진행하세요.

![](https://help.grapecity.com.cn/download/thumbnails/72355636/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1648092580000\&api=v2) 리본 메뉴 모음에서 \[데이터]>\[Excel을 테이블로 가져오기] 선택합니다. &#x20;

![](<../../.gitbook/assets/image (246).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72355636/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1648092580000\&api=v2) \[찾아보기]를 클릭하고 가져올 Excel 파일을 선택한 다음 \[확인]을 클릭합니다. 데이터가 현재 데이터베이스에 저장되는 방법과 위치를 지정할 수 있습니다.   &#x20;

![](<../../.gitbook/assets/image (1587).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72355636/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1648092580000\&api=v2) 가져올 시트 또는 범위를 선택할 수 있습니다.

\[다음]을 클릭합니다.   &#x20;

![](<../../.gitbook/assets/image (1160).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72355636/%E6%AD%A5%E9%AA%A44.png?version=1\&modificationDate=1648092580000\&api=v2) 필드 이름을 설정합니다. 첫 번째 행의 셀 내용을 필드 이름으로 사용하려면 "첫 번째 행에 열 제목 포함됩니다."를 선택합니다.

\[다음]을 클릭합니다.

![](<../../.gitbook/assets/image (498).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72355684/%E6%AD%A5%E9%AA%A45.png?version=1\&modificationDate=1648092581000\&api=v2)가져온 필드의 데이터 형식을 설정합니다. 필드를 선택하여 이름 및 데이터 형식을 수정합니다. 필드를 가져오지 않으려면 필드를 선택한 후 "필드 가져오기 않음(건너뜀)" 선택할 수도 있습니다.

다음을 클릭합니다.      &#x20;

![](<../../.gitbook/assets/image (1032).png>)

선택한 필드의 데이터 형식을 그림 및 첨부 파일로 설정한 경우 폴더의 주소를 설정해야 합니다. 이미지 또는 첨부 파일 필드의 데이터가 절대 경로인 경우 폴더 주소를 설정하지 않을 수도 있습니다. 폴더 주소가 그림 또는 첨부 파일의 데이터와 결합되어 전체 주소로 결합되는 한 지정된 파일에 액세스할 수 있습니다.

첨부 파일 필드의 데이터에는 여러 첨부 파일이 있을 수 있으며 여러 첨부 파일 이름 사이에 "|"가 있을 수 있습니다. "양식 1.csv| 양식 2.csv"과 같이 구분됩니다.&#x20;

![](<../../.gitbook/assets/image (195).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72355684/%E6%AD%A5%E9%AA%A46.png?version=1\&modificationDate=1648092581000\&api=v2) 테이블 이름을 설정하고 \[마침]을 클릭합니다. &#x20;

![](<../../.gitbook/assets/image (1533).png>)
