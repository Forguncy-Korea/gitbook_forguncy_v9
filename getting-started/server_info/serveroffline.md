# 서버(Server) 오프라인 라이선싱

특정 개발 환경에 따라서 인터넷 연결이 없는 환경에서 오프라인 라이선스 인증이 필요하신 경우, 아래와 같이 추가적인 방법을 통해서 라이선스 활성화/비활성화를 진행할 수 있습니다.

{% hint style="info" %}
**오프라인 라이선싱 주의사항**

\* 기업 내부 정책으로 인하여, **외부로의 인터넷 연결이 어려운 경우(내부망)와 같은 부득이한 상황을 제외하고는 오프라인 인증은 권장 드리지 않습니다.**
{% endhint %}

### 오프라인 라이선스 활성화

**1 . Forguncy 서버 관리자를 실행한 후, \[지원]>\[라이선스 목록]에 들어갑니다. 라이선스 목록에 체험판으로 나오는 것을 확인할 수 있습니다. 라이선스 인증을 하기 위해 \[서버 라이선스 추가]버튼을 클릭합니다.**

<figure><img src="../../.gitbook/assets/image (2073).png" alt=""><figcaption></figcaption></figure>

**2.** 아래와 같은 서버 키 활성화 창이 나타나면, 메시어스로부터 구입하신 라이선스 키를 입력해주세요.

<figure><img src="../../.gitbook/assets/image (2074).png" alt=""><figcaption></figcaption></figure>

3 . **컴퓨터가 인터넷에 연결되지 않았다면 아래와 같이 “오프라인 활성화” 버튼이 나타나게 되며,** 해당 버튼을 클릭하여 다음 단계로 진행합니다.

![](<../../.gitbook/assets/image (327).png>)

4 . 아래와 같은 **“오프라인 활성화”** 창에 나타나는 **“오프라인 정품 인증 키”를 아래와 양식에 맞추어 메일로 보내주세요.**

* 메일 주소: [**sales-kor@mescius.com**](mailto:sales-kor@grapecity.com)
* 오프라인 라이선스 활성화 요청 정보
  * 구매(주문)번호: <mark style="color:blue;">**구매 메일에 받으신 주문번호(ex. GCK-000000)**</mark>
  * 기업명:&#x20;
  * 담당자명:
  * 전화번호:
  * 포건시 빌더 라이선스 키: <mark style="color:blue;">**구매 메일에 받으신 포건시 빌더 라이선스 키**</mark>
  * 오프라인 정품 인증 키: <mark style="color:blue;">**현재 오프라인 활성화 창에 보이는 오프라인 인증**</mark>&#x20;

{% hint style="info" %}
**주의: 창을 닫지 마세요!!**\
**메시어스와**의 오프라인 인증 절차가 완료될 때 까지, 아래의 "오프라인 활성화" 창은 반드시 열어 두셔야 합니다.
{% endhint %}

![](<../../.gitbook/assets/image (1138).png>)

5\. 이후 메시어스 코리아에서 보내주신 정보 확인 후에 **"코드 활성화 키"** 값을 보내드립니다.

6\. 메시어스로 부터 받으신 키를 **위 그림의 “코드 활성화” 부분에 붙여 넣으신 후 “확인” 버튼을 누르면 정상적으로 제품이 오프라인 인증**이 됩니다.



### 오프라인 라이선스 비활성

포건시 서버(Server)를 삭제하는 경우, 컴퓨터를 포맷하는 경우, 또는 다른 컴퓨터로 이동할 경우, <mark style="background-color:orange;">**구매하신 라이선스를 계속 유지하기 위해서는 라이선스 인증을 해제하는 작업을 반드시 해야 합니다.**</mark>

{% hint style="info" %}
#### 주의사항

<mark style="color:red;">**\*삭제/포맷 시에 라이선스를 비활성화를 하지 않아, 소실된 라이선스에 대해서 복구가 어려움을 안내 드립니다.**</mark>
{% endhint %}



**1. Forguncy 서버관리자를 실행 후, “지원">"라이선스 목록"을 선택합니다.**

<figure><img src="../../.gitbook/assets/image (2080).png" alt=""><figcaption></figcaption></figure>

**2.** **"비활성화"를 클릭합니다.**

![](<../../.gitbook/assets/image (179).png>)

**3. “Forguncy 서버 인증 비활성화” 창에서 “비활성화” 버튼을 클릭합니다.**

![](<../../.gitbook/assets/image (1026).png>)

**4. 해당 컴퓨터에가 오프라인 상태인 경우 “오프라인 비활성화” 확인창이 나타납니다.**

![](<../../.gitbook/assets/image (1278).png>)

**5. 아래와 같은** **“오프라인 활성화”** 창에 나타나는 **“오프라인 비활성화의 응용 프로그램 키” 를 아래와 양식에 맞추어 메일로 보내주세요.**

* 메일 주소: [**sales-kor@mescius.com**](mailto:sales-kor@grapecity.com)
* 오프라인 라이선스 활성화 요청 정보
  * 구매(주문)번호: <mark style="color:blue;">**구매 메일에 받으신 주문번호(ex. mesius-000000)**</mark>
  * 기업명:&#x20;
  * 담당자명:
  * 전화번호:
  * 오프라인 비활성화의 응용 프로그램 키: <mark style="color:blue;">**현재 오프라인 비활성화 창에 보이는 오프라인 키**</mark>

{% hint style="info" %}
**주의: 창을 닫지 마세요!!**\
그레이프시티와의 오프라인 비활성화 절차가 완료될 때 까지, 아래의 "오프라인 비활성화" 창은 반드시 열어 두셔야 합니다.
{% endhint %}

![](<../../.gitbook/assets/image (87).png>)

**6. 메일을 보내주신 후에, "닫기" 눌러 라이선스 비활성화를 종료합니다.**<br>

**7.  아래와 같이 라이선스 목록에 체험판이라고 나타나면 정상적으로 비활성화된 것 입니다.**
